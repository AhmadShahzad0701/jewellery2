--------------------------------------------------
HOW TO RUN BACKEND
--------------------------------------------------
1. Install Java JDK 19

2. Set JAVA_HOME (Windows PowerShell):
   $env:JAVA_HOME="C:\Program Files\Java\jdk-19"

3. From project root directory, run:
   .\mvnw.cmd spring-boot:run

Backend will run on:
http://localhost:8080

--------------------------------------------------
HOW TO RUN FRONTEND
--------------------------------------------------
1. Navigate to client folder:
   cd client

2. Install dependencies:
   npm install

3. Run frontend server:
   npm run dev

Frontend will be available at:
http://localhost:5173

--------------------------------------------------
NOTES
--------------------------------------------------
- Make sure backend is running before using frontend
- MongoDB must be available for full functionality
