## üß† Software Reengineering Focus

This project emphasizes:
- Identification and removal of **code smells**
- Application of **refactoring techniques**
- Improved **separation of concerns**
- Clean architecture principles
- Better **maintainability and scalability**

### Key Improvements
- Thin Controllers (business logic moved to services)
- Removal of duplicated code using Extract Method
- Introduction of DTOs to eliminate primitive obsession
- Strategy Pattern implemented for product sorting
- Defensive programming for safer runtime behavior
